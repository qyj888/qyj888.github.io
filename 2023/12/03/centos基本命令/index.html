centos基本命令

2023-12-03
本文总阅读量次

一、centos基本命令

ls文件目录

ls / 根目录

ls -a 查看隐藏文件

隐藏文件 .xxxxxx

​ xxxxx

ls . 当前目录

ls .. 上级目录

touch 文件名 创建文件（按ls查看）

touch .文件名 创建隐藏文件（按ls -a查看）

ls -al 显示当前目录下的所有的文件及属性

第一列是文件属性

mkdir 创建文件夹

mkdir -p 文件名/文件名/文件名创建递归文件夹

cd切换目录

whoami查看当前用户的名字

cat 文件名 查看文本文件

id当前用户的权限

pwd当前所在的目录

rm -f 文件名 强制删除

rm -rf test1/ 强制删除文件夹下所有的内容

ls /bin |grep mv查找bin目录所有带mv的文件

ls /bin|more

ls /bin|less

ls /bin|grep ls

cat /bin/ls

vi 文件名 编辑文件

touch anaconda-ks.cfg

cat -n 文件名 查看文件内容（-n显示行号）

chmod 755 文件名 修改权限

echo “123“(想要写的内容) >>test1/4 将内容追加到test1目录中的4里面

find /bin -name ls搜索在bin目录下名字为ls的文件

find /bin -user root

find /bin size +204800 搜索bin目录中大小在100M以上的文件

find /bin -iname ls

useradd admin 创建用户

passwd admin 修改admin密码（不加用户名，就是修改当前用户密码）

ip add显示centos的ip地址

二、物理机连接centos

1.systemctl enable sshd

2.systemctl start sshd

3.vi /etc/ssh/sshd_config

4.按esc，输入：set nu显示行号，找到38行去掉#后面没有yes就添加一个yes即可，PermitRootLogin yes

5.systemctl restart sshd

6.ip add 查找centos的地址

7.ssh root@192.168.102.137
