一、字符集和编码

ascii =>编排了128个文字符号，只需要7个0和1就可以表示了

1 byte =>8bit

ANSI => 一套标准，每个字符16bit，2byte

gb231编码(中国),gbk编码(windows 默认就是这个) ，16bit，2byte

JIS编码(日本)

Unicode:万国码，32bit，4byte（只是一个标准，不能用）

utf：是可变长度的unicode，可以进行数据的传输和存储->行书，草书，隶书

btf-8:最短的字节长度8

​ 英文：8bit，1byte

​ 欧洲文字：16bit，2byte

​ 中文：24bit，3byte

gbk和utf-8不能直接进行转化

三、运算符

算数运算 + - * / % //

比较运算符 > >= < <= == !=

赋值运算符 = += -= *= ….

逻辑运算符

and 并且，左右两端同时成立结果为真

or 或者，左右两端有一个成立结果就成立

not 非，非真即假，非假即真

当and和or以及not同时出现的时候，最好加上括号，不会产生歧义或者不易理解

运算顺序：括号>not>and>or

成员运算符

in 判断xxx是否在xxx中出现了

not in 判断xxx是否不在xxx中出现了
