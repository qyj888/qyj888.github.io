二、picgo和typora的连接

1.打开typora,选择文件->偏好设置

2.点击图像,插入图片时选择第三个,然后勾选前四个,上传服务选择PicGo(app),PicGo路径精确定位到PicGo.exe

3.在github上点击个人主页,选择setting

4.找到develoer setting->Personal access tokens->Tokens(classic)->Generate new token->Generate new token(classic),命名为123,生效时间选择永久,勾选第一个,生产令牌后复制令牌

5.打开PicGo,点击图床设置->GitHub,然后编辑

6.图床配置名为123,仓库名为github用户名/仓库名,分支名设为main,Token处填写刚刚复制的令牌,设定存储路径为img/,点击确定,然后设为默认图床

7.回到typora,点击验证图片上传选项,会显示验证成功,这样PicGo和typora的连接就建立成功了.

flash钓鱼

一、flsah页面制作

1.在虚拟机上下载PHPstudyhttps://www.xp.cn/

2.将Apache2.4.39启动

3.点击网站->管理->打开根目录

4.删除里面所有的文件，然后创建一个txt后缀的文本文件，然后重命名为html后缀的index文件

5.在物理机上创建一个同样名字的HTML文件，再用物理机打开vscode

6.点击文件->打开文件

7.选择创建好了的html后缀的文件，然后点击打开，按住shift+!再回车就会出现主体代码

8.在body部分输入

9.把刚刚在win10虚拟机中创建的html后缀的文件以记事本方式打开

10.把刚刚在vscode里面编辑的代码全选放到虚拟机打开的记事本中并保存

11.使用一个js脚本，放在phpstudy的根目录下面

📎flash.js

12.打开phpsutdy,点击网站->管理->打开网站

13.打开之后就会显示出已经成功做好的flash页面

二、flash捆绑木马

1.关掉所有的杀毒软件，然后下载好一个木马程序和flash安装包

📎木马.rar

2.同时选中木马程序和flash安装包，右击，然后选择添加到压缩文件

3.勾选创建自解压格式压缩文件

4.点击高级->自解压选项

5.解压路径设置 C:\windows\temp

6.然后点击设置选项卡，在解压后运行里面输入

C:\windows\temp\flashcenter_pp_ax_install_cn.exe

（黄色背景部分对应之前下载好的flash安装包的名字）

C:\windows\temp\flash.exe

7.点击模式选项卡，选择全部隐藏

8.点击更新选项卡，将更新模式和覆盖模式都选择为第二个

9.点击确定，然后会显示出一个压缩包

10.下载一个图标更改程序

📎更改图标.rar

11.点击Restorator.exe启动程序

12.将刚刚做好的压缩包和flash安装包拖到资源树中

13.删除压缩包的图标，将flash安装包的图标复制到压缩包中git clone

14.再次点击压缩包的图标，可以发现它变成了flash安装包的图标

15.点击保存

16.将改好的压缩包放入phpstudy的根目录中，并将其名字改为flash压缩包的名字

17.打开浏览器，在地址栏输入127.0.0.1，就可以看到捆绑木马的flash网站做好了

18.点击立即升级，下载压缩包并打开，打开之后可以看到木马已经植入电脑

三、自制的其它钓鱼网页（不带免杀）

1.打开一个steam网页，右击选择另存页面为，将源码保存phpstudy的根目录下

2.找到一个steam安装包

3.选中之前下好的木马和steam安装包，右击重复之前的压缩步骤，这一步的路径要改为steam安装包的名字

4.压缩好了之后，再重复之前更改图标的操作，然后将压缩包放入phstudy的根目录中

5.把之前下好的Sign In.htm改为index.html

6.右击index.html用记事本打开，找到installsteam这一部分

7.将href=” “里面的内容改为/Steaminstaller.exe ,保存更改

8.打开浏览器在地址栏输入127.0.0.1，点击Install steam

9.打开下载好的文件，可以看到电脑跟之前已植入木马

一、简单的钓鱼网站的使用

1.在kali中打开终端，然后输入vim /etc/proxychains.conf

2.将打开的文件的最后一行改为代理IP和端口号

3.然后按ESC输入冒号：，再输入wq！保存并退出编辑

4.在终端上输入proxychains git clone https://github.com/makdosx/mip22

5.输入cd mip22

6.输入chmod +x ./*

7.输入proxychains -q bash ./mip22.sh，然后会出现以下界面

8.输入1

9.选择一个钓鱼内容，例如输入51

10.输入3，等待加载

11.可以得到三个url，选择https的url，复制用浏览器打开，可以得到一个steam登录的钓鱼页面

二、HTTP动态码的获取

1.下载VScodehttps://code.visualstudio.com/

2.在VScode的扩展商店里面搜索python，安装第一个

3.在VScode的扩展商店里面搜索code run，安装第一个

4.新建一个python类型的文件，输入代码

5.将输入的python代码保存在已经安装好的python目录下

6.在搜索栏中搜索cmd，打开cmd

7.输入python 1.py

8.出现报错，根据报错信息下载插件，输入pip install requests -i http://mirrors.aliyun.com/pypi/simple/ –trusted-host mirrors.aliyun.com

9.再次输入python 1.py进行尝试，看是否还缺少插件

10.再次出现报错，根据报错信息安装插件，输入pip install chardet -i http://mirrors.aliyun.com/pypi/simple/ –trusted-host mirrors.aliyun.com

11.再次输入python 1.py进行尝试，可以发现成功获取动态码

三、大灰狼的使用

1.关闭物理机和win10虚拟机的防病毒及防火墙，下载大灰狼并打开

2.点击服务生成

3.将域名ip改为自己的本机ip，然后点击生成服务

4.生成一个后缀为.exe的可执行文件，将文件拖入win10虚拟机中，在win10虚拟机上双击执行，然后就可以用大灰狼对虚拟机进行远程操作了。

5.右击已连接的虚拟机，可以对win10虚拟机进行远程控制，键盘记录等
